数学王子高斯

【摘要】高斯被誉为“数学王子”。

数学王子：高斯

高斯（Carl Friedrich Gauss），1777.4.30出生于德意志布伦瑞克。高斯是贫穷人家的子弟。1855.2.23凌晨于哥廷根去世。

主要成果：

1.最小二乘法——高斯马尔科夫定理

最小二乘法，又称最小平方法，是一种数学优化技术。它通过最小化误差的平方和寻找数据的最佳函数匹配。

高斯马尔科夫定理：在线性回归模型中，如果误差满足零均值、同方差且互不相关，则回归系数的最佳线性无偏估计(BLUE, Best Linear unbiased estimator)就是普通最小二乘法估计。

线性函数模型
简单线性回归模型
多元线性回归模型

2.质数定理

素数定理描述素数的大致分布情况。
对正实数x，定义π(x)为不大于x的素数个数。
π(x)≈x/ln x 其中ln x为x的自然对数。当x趋近∞，π(x) 和x/ln x的比趋近1

3.高斯钟形曲线（正态分布曲线）

并非由高斯最初发明。

4. 17边形

正十七边形，内角和为2700°。高斯用尺规作图画出了正十七边形。

7.三角形全等定理

8.测量平差理论

测量平差就是指在测量中对测量数据进行调整以求得最接近真实值的最优解的方法。主要使用最小二乘法

9.小行星谷神星的运行轨迹

10.复活节日期的计算公式

在一定范围内，投影面上任何点上两个微分线段组成的角度投影前后保持不变的一类投影
